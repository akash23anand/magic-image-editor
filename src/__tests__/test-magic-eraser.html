Magic Eraser Tool Test

Test Overview

This test verifies that the magic eraser tool workflow is working correctly:

  1. User clicks Magic Eraser button
  2. Mask drawing overlay appears
  3. User can draw mask with adjustable brush
  4. User clicks Apply to process erasure
  5. Mask is passed to UnifiedAIService
