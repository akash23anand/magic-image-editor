# Demo Images for Layer Anything

This directory contains sample images for the Layer Anything demo application.

## Image Requirements

For the demo to work properly, place the following images in this directory:

1. **document-sample.jpg** - A document with mixed text and graphics
   - Recommended size: 800x600 or similar
   - Should contain clear, readable text
   - Good for demonstrating OCR capabilities

2. **portrait-sample.jpg** - A portrait photo
   - Recommended size: 600x800 or similar  
   - Clear subject against background
   - Good for background removal demos

3. **scene-sample.jpg** - A complex scene with multiple objects
   - Recommended size: 800x600 or similar
   - Multiple distinct objects
   - Good for object detection and segmentation

4. **mixed-sample.jpg** - Mixed content with text and objects
   - Recommended size: 800x600 or similar
   - Combination of text and visual elements
   - Good for comprehensive layer extraction

## Image Sources

You can use:
- Stock photos from Unsplash, Pexels, or similar free sources
- Screenshots of documents or web pages
- Your own photos (ensure they meet the requirements above)

## File Formats

- Supported: JPG, PNG, WebP
- Recommended: JPG for photos, PNG for documents/screenshots
- Keep file sizes reasonable (< 2MB each) for demo performance

## Demo Functionality

Each image type demonstrates different Layer Anything capabilities:

- **Document**: OCR text extraction, text layer creation
- **Portrait**: Background removal, person segmentation  
- **Scene**: Object detection, multi-object segmentation
- **Mixed**: Combined text and object processing

Place your images in this directory and the demo will automatically use them.
